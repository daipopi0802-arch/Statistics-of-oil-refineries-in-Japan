# Statistics of oil refineries in Japan
日本国内の製油所の処理能力、系列ネットワーク、災害時の停止シナリオなどをシミュレーション・視覚化する統合GISコマンドセンター（ダッシュボード）

## 概要
このプロジェクトは、日本国内の主要な製油所に関する公開統計データを基に、Leaflet.js を用いて地図上に可視化したインタラクティブなWebアプリケーションです。
### 機能

- **拠点可視化**: 各製油所の処理能力に応じたマーカー表示と、夜間・昼間・衛星マップの切り替え
- **需給カバーゲージ**: 日本の石油需要（約320万BD）に対する、稼働中拠点の総処理能力カバー率をリアルタイム計算
- **停止シナリオシミュレーター**: 南海トラフ地震、大型台風、有事（マラッカ海峡封鎖）を想定した、被影響拠点の自動特定と残存能力のシミュレーション
- **系列ネットワーク**: ENEOS、出光興産、コスモ石油など、各系列ごとの拠点ネットワーク描画
- **航路シミュレーション**: 中東から各製油所までのマラッカ海峡経由のタンカー航路・所要日数の試算
- **歴史タイムライン**: 1955年から2026年までの、日本の石油精製産業の変遷と拠点数の推移を追体験
- 
## 使用技術
- HTML5 / CSS3
- Tailwind CSS (スタイリング)
- Leaflet.js (地図マッピング)
- Chart.js (データグラフ化)
